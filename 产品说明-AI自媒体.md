## 素材筛选原则

### 三维评分

从 `ai_discovery` 中筛选素材时，综合三个维度：

| 维度 | 权重 | 说明 |
|------|------|------|
| importance | 高 | `high` 优先，`medium` 需其他维度补偿 |
| user_note | 高 | 有正面备注的优先（如"非常喜欢"、"确实有用"）；有负面备注的降权或跳过 |
| 时效性 | 中 | `discovered_at` 或 `published_at` 越近越好；超过 7 天的需要有"常青"价值才选 |

### 二次验证（必须）

筛选出候选素材后，**必须通过以下方式验证**再决定是否使用：

1. **时效验证**：WebSearch 搜索该话题，确认事件没有后续反转、新进展、或已过时
2. **热度验证**：检查原帖当前的互动数据（likes/retweets/comments），确认不是虚假热度
3. **独特性验证**：搜索公众号/抖音是否已有大量同主题内容，避免跟在别人后面写

验证不通过则跳过该素材，选下一个。

## 内容加工规范

### 写作风格

- **不是翻译/搬运**：DB 中的 summary/md 是原始素材，需要重新组织和加工
- **说人话**：技术概念用类比解释，不堆术语
- **有观点**：不只是"发生了什么"，更要说"这意味着什么"、"对你有什么影响"
- **有结构**：标题 → 一句话钩子 → 正文 → 行动建议/思考题收尾

### 排版规范

- 字体：正文首选「平方」，备选系统默认无衬线
- 段落短，适合手机阅读（每段 2-4 行）
- 关键信息加粗，核心观点用引用块
- 配图标记：`![描述](URL)`

### 篇幅

| 栏目 | 字数 |
|------|------|
| AI 周报 | 1000-1500 字 |
| 深度拆解 | 1500-2500 字 |
| 产品灵感 | 800-1500 字 |

## 单轮内容生产流程

> 触发方式：用户说「写一篇 AI 博文」或「从 DB 选题写一篇」

### Step 1：选题

1. 查询 DB 候选素材：
   ```sql
   SELECT id, title, category, importance, user_note, discovered_at, length(md) as md_len
   FROM ai_discovery
   WHERE importance IN ('high','medium')
   ORDER BY
     CASE importance WHEN 'high' THEN 1 ELSE 2 END,
     discovered_at DESC
   LIMIT 20
   ```
2. 按「三维评分」筛出 Top 3 候选
3. 执行「二次验证」
4. 向用户推荐验证通过的最佳选题（附理由），等用户确认

### Step 2：加工

1. 读取选定素材的完整 md 和 raw_data
2. 确定栏目（周报/深度拆解/产品灵感）
3. 按写作风格 + 排版规范生成博文初稿
4. 输出给用户审核

### Step 3：用户审核

- 用户确认或给修改意见
- 修改后再次确认
- 确认通过后进入发布

### Step 4：发布到微信公众号

> 公众号后台各页面 URL 含会话 token（每次登录会变），不可写死。通过页面交互导航进入。

**后台导航参考**（通过 MCP Playwright 页面交互到达）：
- 草稿箱：首页 → 内容与互动 → 草稿箱
- 素材库：首页 → 内容与互动 → 素材库
- 文章模板：首页 → 内容与互动 → 文章模板（可创建复用模板）
- 发表记录：首页 → 内容与互动 → 发表记录
- 原创管理：首页 → 内容与互动 → 原创
- 合集管理：首页 → 内容与互动 → 合集（用于栏目分类，后续建立）

**创建文章**：鼠标移到「新的创作」按钮 → 出现下拉菜单 → 点击「写新文章」→ 打开编辑器 Tab

**发布流程**：
1. MCP Playwright 打开 mp.weixin.qq.com
2. 复用共享 Chrome 登录态
3. 通过「新的创作 → 写新文章」进入编辑器
4. 填入标题 + 内容（排版见下方规范）
5. 预览截图给用户确认
6. 用户确认后发布（或保存草稿）
7. 截图保存到 `.screenshots/`

**排版规范**：
- 公众号编辑器不支持原生 Markdown，需转换为富文本
- 字体：正文默认「平方」（iOS 原生支持），安卓侧选择优雅的无衬线回退字体
- 段落短（每段 2-4 行），适合手机竖屏阅读
- 关键信息加粗，核心观点用引用块样式
- 发布前必须手机端预览测试排版效果

### Step 5：发布到抖音（图文）

> 公众号发布完成后，**内容需适配抖音图文格式**再发布

1. 将公众号长文拆分为 5-10 张图文卡片的文案
2. 每张卡片一个核心信息点，文字精简
3. MCP Playwright 打开抖音创作者平台
4. 选择「图文」发布模式
5. 填入封面标题 + 正文
6. 发布或保存草稿
