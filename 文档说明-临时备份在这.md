## AI Skill/Prompt 工程

Claude Code Skill 海报生成（ethanweber/posterskill，393 stars）：Meta 研究员 Ethan Weber 做的，用 Claude Code Skill 读取论文源码后生成一个单文件 React HTML 学术海报。关键不在海报本身，在交互模式——生成的 HTML 支持在浏览器里拖拽调整布局，调好后点一下导出 JSON，把 JSON 粘贴回 Claude，Claude 按你的布局重新生成代码。人负责空间感，AI 负责内容，通过 JSON 来回传递。这个模式可以二次进化成国内自媒体工具（小红书图文卡片、公众号长图文等），只需要替换模板和 prompt，拖拽渲染逻辑不用动。

Claude Code Skill 自进化方法（基于 Anthropic Skill-Creator 2.0，2026-03-03 发布）：本质是 TDD 思想用在 prompt 上。先给 skill 写 15-30 个测试用例（输入 + 通过条件），跑一遍算通过率，AI 自动分析失败用例的共性，然后自动改 skill 的 prompt 指令，再跑测试，重复几轮。有人实测 4 轮从 56% 提升到 92%。实用场景：以后给 Onemake 写的任何 skill（文案生成、changelog 等）都可以用这个方法量化迭代，不用凭感觉调 prompt。
