# 热梗公众号：采集与发布流程

> 从各渠道采集当前热门梗/热点话题，加工为适合公众号的图文内容，通过 MCP Playwright 发布到微信公众号。

## 目标用户

- 15-30 岁追热点、玩梗的年轻用户
- 转发有趣内容到群聊/朋友圈的用户

## 采集平台与优先级

| 平台 | 入口 | 优先级 | 说明 |
|------|------|--------|------|
| 搜狗微信搜索 | weixin.sogou.com | 均等 | 公众号生态内的热梗文章。⚠️ 不支持按时间筛选，需用具体梗名搜索 |
| 微博热搜 | weibo.com/hot/search | 均等 | 实时性最强，热梗发源地 |
| 抖音热榜 | douyin.com | 均等 | 看标题+评论区，**不看视频** |
| B站热门 | bilibili.com/v/popular | 均等 | 看标题+评论区+弹幕热词，**不看视频** |

> 四个平台各占 ~25% 采集时间。

## 数据存储

复用 `ai_discovery` 表，以下字段约定：

| 字段 | 热梗场景取值 |
|------|-------------|
| `platform` | `sogou_wechat` / `weibo` / `douyin` / `bilibili` |
| `category` | `hot_meme`（固定） |
| `importance` | `high` = 全网爆梗 / `medium` = 平台热门 / `low` = 小众但有趣 |
| `raw_data` | JSON：`{ text, comments[], images[], heat_score, source_url }` |
| `md` | 加工后的公众号文章内容（Markdown） |

其余字段（`url`、`author`、`title`、`summary` 等）含义不变。

## 单轮采集流程

### Step 1：准备

1. 读取本文档 + Playwright 操作规范
2. 查 `ai_discovery` 中 `category = 'hot_meme'` 的已有 `url` 和 `title`，用于去重
3. 注入反检测脚本（`addInitScript`）

### Step 2：浏览采集

按优先级依次访问各平台，像一个追热点的年轻人一样浏览。

**搜狗微信搜索（主力）：**
- 搜索当前热门关键词/梗名
- 提取公众号文章标题、摘要、来源公众号、发布时间
- 点进高热度文章获取完整内容和配图

**微博热搜：**
- 浏览热搜榜单，关注有梗属性的话题（非纯新闻）
- 提取话题名、热度值、代表性评论
- 筛选标准：有趣/有梗/有传播性，排除纯时政/灾难新闻

**抖音热榜：**
- 浏览热榜标题和评论区
- 提取标题、热度、精选评论（尤其是玩梗评论）
- **不打开视频**，仅从标题和评论理解梗的内容

**B站热门：**
- 浏览热门视频列表
- 提取标题、播放量、弹幕热词、精选评论
- **不打开视频**，从标题+评论+弹幕理解内容

**图片采集：**
- 采集过程中遇到适合配图的图片（表情包、梗图、截图），保存 URL 到 `raw_data.images[]`
- 优先采集：已广泛传播的梗图、评论区高赞截图、热搜相关配图

### Step 3：入库

- `INSERT OR IGNORE` 写入 `ai_discovery`，`category = 'hot_meme'`
- URL 从页面元素 href 提取，禁止拼接
- `raw_data` 尽量丰富：原文/评论/图片 URL/热度数据

### Step 4：生成公众号文章（md 字段）

每条入库的热梗，生成一篇适合公众号的短文章，写入 `md` 字段。

**文章风格：**
- 标题醒目、有梗味（可用 emoji 但不过度）
- 正文简短有力，一般 300-800 字，热梗不需要长篇大论
- 用年轻人的语气，不要官媒腔
- 配图位置用 `![描述](图片URL)` 标记

**排版规范：**
- 字体：正文首选「平方」，备选系统默认无衬线字体（苹方/思源黑体）
- 标题：加粗加大，视觉突出
- 段落间留白充足，适合手机阅读
- 关键梗/金句用引用块或加粗高亮
- 整体风格：干净、醒目、易扫读

**内容结构（灵活调整，不必每篇都有）：**
1. 梗的由来 / 事件起因（1-2 句）
2. 梗的内容 / 精华评论 / 经典用法
3. 配图（如有）
4. 一句话点评或神回复收尾

### Step 5：结束

输出本轮采集总结：采集了哪些梗、各平台热度概览。

## 发布流程

> 触发方式：用户说「根据 #id 去微信公众号发布一下」

### Step 1：读取内容

从 `ai_discovery` 读取该 `id` 的 `title` 和 `md` 字段。

### Step 2：MCP 连接公众号后台

1. 通过 MCP Playwright 打开微信公众号后台（mp.weixin.qq.com）
2. 用户已在共享 Chrome 中登录，直接复用登录态
3. 进入「图文消息」编辑器

### Step 3：填写内容

1. 填入标题
2. 将 md 内容转换为公众号编辑器格式粘贴
3. 插入图片（如 raw_data 中有图片 URL）
4. 预览确认排版

### Step 4：发布

- 点击发布（或保存草稿，等用户确认后发布）
- 截图保存发布结果到 `.screenshots/`

## 启动命令

```

```

